# EasyDeals
Easy deals provide a user friendly environment to make deals with other users in the website. The system consists of a registration page where students need to register to access the website. The students can make deals by posting the deals from their registered profile. The deals page provides all the deals available and the user can make a deal using the chat application built in the system. Once the deal is confirmed the post will be deleted automatically.
